计算属性（computed）和侦听（watch）

watch执行异步请求示例，音乐搜索

总结一下：computed和watch的区别：

1.computed根据现有数据，计算出新的结果

2.watch没有返回值，watch可以进行异步操作，computed不能！！！

3.watch可以侦听现有的data和computed的数据改变，进而执行相应的函数
